# Тестовое задание

## Запуск проекта

Запуск проекта в режиме разработки:

```bash
npm run dev
# or
yarn dev
```

Проект будет доступен по ссылке [http://localhost:3000](http://localhost:3000) в вашем браузере.

Сборка и запуск проекта в продакшн режиме:

```bash
npm run build
# or
yarn build
```

затем

```bash
npm run start
# or
yarn start
```
